Case Creation Tips

Naming Convention

Use descriptive names that include date, incident type, or ticket number for easy identification.

File Organization

All uploaded files will be associated with this case. Sigma and Chainsaw rules will be applied automatically to analyze events.

Collaboration

Team members can contribute files and analysis to the case. All activities are logged for audit purposes.

Search & Analysis

Once files are uploaded, you can search through events, view rule violations, and analyze patterns using the built-in search interface.
